# AuroraGroundControl
`droneEmulator` - В этой папке находится **собственноручно разработанный эмулятор дрона**, написанный в виде скетча. Он полностью симулирует поведение реального беспилотного летательного аппарата (БПЛА) по протоколу **MAVLink** — на ESP32 стандарту де-факто в области взаимодействия наземных станций и автопилотов.

В рамках подготовки к интеграции MAVSDK мы выполнили экспериментальную сборку библиотеки `libmavsdk_arm32.so` под 32‑битную архитектуру ARMv7.
